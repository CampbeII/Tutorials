# Logging
```sh
%systemRoot%\System32\Logfiles
```

## Types

System Logs:
- system components
- driver failure
- resource conflicts
- hardware issues

Application Logs:
- Software related errors
- Failing to connect to db
- Bottlenecks

Security Logs:
- Authentication attempts
- Right assignments
- Policy changes

Forwarded Event Logs:
- Assembled from other environments
- Used to analyze networked environments
